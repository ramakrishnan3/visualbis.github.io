Who should read this guide?¶

This User Guide for valQ as Custom Visual for Microsoft Power BI offered by Visual BI is meant for users that are going to use Apps leveraging the valQ Custom Visual to analyse value driver relationships, compare versions such as Forecast vs Current Budget and simulate what*if scenarios such a change in price, volumes and efficiencies real*time.

What is the Visual BI valQ for Power BI as a Custom Visual Component?¶

valQ for Microsoft Power BI is a custom visual component which involves mathematical or conceptual business models visualized in a way that links the business KPI’s (what management or stakeholders care about) to the operational drivers (the things that can be influenced to change the KPI’s).
